For Running the project via tcl script:
1. Open Vivado TCL Console.
2. cd to the directory "work" inside fpga folder.
3. Then, run "source ../bram.tcl" to create the project.
4. For running SDK, export hardware with selecting "SDK" folder instead of "Local to repository".
5. Same with the Launching SDK step, choose folder "SDK" instead of "Local to repository".
